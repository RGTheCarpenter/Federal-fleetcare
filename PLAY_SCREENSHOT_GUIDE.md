# Play Store Screenshot Guide

Use a real Android phone if possible so the screenshots match the app experience people will install.

## Screenshot set

Capture these in portrait orientation:

1. Login screen
   - Show the FleetCare Mobile sign-in view
   - Keep the form clean with no private personal data visible

2. Vehicles dashboard
   - Show the Vehicles tab
   - Include the stats row and vehicle list

3. Tracking status
   - Show the `Tracking status` panel
   - Make sure the status chip and last saved point are visible

4. Capture a location
   - Show the `Capture a location` panel
   - Include the `Use my live location` button

5. Trip tracking
   - Show an active trip if possible
   - Include the active trip status and stop button

6. Trip history and route map
   - Show at least one saved trip with the route map visible

7. Maintenance and attachments
   - Show a maintenance entry with a photo or attachment preview

8. Reports and alerts
   - Show the reminders/alerts area or the PDF report filters

## Clean capture tips

- Use realistic but non-sensitive company data
- Avoid empty screens when possible
- Keep the phone status bar visible
- Do not show browser chrome if you are capturing from the installed Android app
- Use the same branding and sample company name across all screenshots

## Suggested order in Play Console

1. Vehicles dashboard
2. Tracking status
3. Capture a location
4. Trip tracking
5. Trip history and route map
6. Maintenance and attachments
7. Reports and alerts
8. Login screen
